# U-Learning 自动倍速刷课脚本

适用于 [ua.ulearning.cn](https://ua.ulearning.cn/) 平台的油猴（Tampermonkey）脚本，自动倍速播放视频课程，自动跳转下一节，解放双手。

油猴脚本地址：https://greasyfork.org/zh-CN/scripts/577154-%E4%B8%9C%E8%8E%9E%E7%90%86%E5%B7%A5ulearning-%E8%87%AA%E5%8A%A8%E5%80%8D%E9%80%9F%E5%88%B7%E8%AF%BE%E8%84%9A%E6%9C%AC
## 功能特性

### 核心功能

| 功能 | 说明 |
|------|------|
| 自动倍速播放 | 默认 8 倍速，支持 1/2/4/8/16 倍速切换 |
| 自动播放 | 检测到视频后自动播放，被暂停后自动恢复 |
| 自动下一节 | 视频播放完毕自动跳转下一页/下一章 |
| 自动静音 | 避免倍速播放噪音，同时解决浏览器自动播放限制 |
| 跳过已看完 | 检测到进度 >= 95% 自动跳转到下一节 |
| 卡顿修复 | 视频转圈加载时自动微调进度条恢复播放 |

### 防检测机制

| 机制 | 说明 |
|------|------|
| 拦截 `pause()` | 阻止平台程序化暂停视频，从根源防止暂停-播放循环 |
| 恢复冷却 | 3 秒冷却期，避免与平台暂停机制"打架" |
| 倍率锁定 | 防止平台重置播放倍率 |
| AJAX 安全补丁 | 拦截空 JSON 响应，防止 `SyntaxError` 报错 |
| 弹窗自动处理 | 自动关闭"是否在观看"等确认弹窗 |

### 控制面板

右上角可拖拽浮动面板，实时显示：

- 当前倍率（可切换）
- 自动播放 / 自动下一节 / 静音 开关
- 播放进度（`viewProgress` 百分比 + 时间进度）
- 卡顿修复状态

## 安装方法

1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 浏览器扩展
2. 点击 Tampermonkey 图标 → **添加新脚本**
3. 将 `ulearning-autoplay.user.js` 的内容粘贴进去，保存
4. 访问 [ua.ulearning.cn](https://ua.ulearning.cn/) 即可自动生效

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + Shift + U` | 显示/隐藏控制面板 |

## 配置说明

脚本顶部的 `CONFIG` 对象可修改默认配置：

```javascript
const CONFIG = {
    playbackRate: 8,       // 播放倍率（1/2/4/8/16）
    autoPlay: true,        // 是否自动播放
    autoNext: true,        // 是否自动跳转下一节
    autoMute: true,        // 是否自动静音
    skipInterval: 3000,    // 主循环检测间隔（毫秒）
    maxRetry: 10,          // 视频检测最大重试次数
    debug: false,          // 调试模式（开启后控制台输出详细日志）
};
```

## 工作原理

### 自动跳转策略（按优先级）

1. **精确选择器** — `span[data-bind*="nextPage"]`、`button[data-bind*="goNextPage"]` 等
2. **文本匹配** — 查找包含"继续下一章""下一章""下一页""下一节"的按钮（排除"上一页"等）
3. **侧边栏目录** — 查找当前激活项，点击下一个
4. **弹窗确认** — 点击弹窗中的"继续""下一"按钮

### 已看完检测

1. 读取页面 `viewProgress` 数值（如 `38.4`），`>= 95` 视为已看完
2. 检测 DOM 中的"已看完"文本
3. 检测进度条宽度 `>= 95%`

### 卡顿检测与修复

- 优先通过 `viewProgress` 数值判断：6 秒内数值未增加 → 判定卡住
- 无 `viewProgress` 时降级为 `currentTime` 检测
- 修复方式：微调 `currentTime += 0.5s`（模拟手动拉进度条）
- 最多尝试 5 次，超过则自动跳转下一节

## 注意事项

- 部分课程如果有防作弊弹窗答题，需要手动处理
- 用户手动点击暂停后，脚本不会自动恢复，需点击面板上的"恢复自动播放"按钮
- 建议倍速不要设置太高（8-16 倍），部分视频服务器可能限制下载速率
